# Contribuer à Prix Essence Québec

Merci de votre intérêt pour ce projet! Toute contribution est la bienvenue.

## Licence

En contribuant à ce projet, vous acceptez que vos contributions soient
publiées sous la licence MIT décrite dans le fichier [LICENSE](LICENSE).

## Utilisation d'outils d'IA

L'utilisation d'assistants d'intelligence artificielle (GitHub Copilot,
Claude, ChatGPT, etc.) est permise pour contribuer à ce projet, aux
conditions suivantes :

1. **Responsabilité humaine.** Le contributeur demeure responsable de tout
   code soumis, qu'il soit écrit manuellement ou avec l'aide d'une IA.
2. **Transparence.** Si une contribution a été substantiellement générée par
   une IA, merci de l'indiquer dans le message de commit ou la pull request
   (par exemple : `assisté par IA` ou `ai-assisted`).
3. **Révision.** Tout code généré par IA doit être relu, compris et testé
   par le contributeur avant soumission.
4. **Respect des licences.** Le contributeur s'assure que le code soumis ne
   reproduit pas de code tiers sous licence incompatible.

## Soumettre une contribution

1. Forkez le dépôt.
2. Créez une branche descriptive (`feat/ma-fonctionnalite`).
3. Commitez vos changements avec des messages clairs.
4. Ouvrez une pull request en décrivant les modifications apportées.
